FONCTIONNALITES AJOUTEES (v14.7.0)
----------------------------------

1. AFFICHAGE DE LA LISTE DES ELEMENTS/SOUS-ELEMENTS LIES
   Dans le menu d'edition (section Proprietes), sous la mention 
   "Cet element est lie" ou "Ce sous-element est lie", 
   on affiche maintenant :
   - La liste complete des elements/sous-elements lies
   - Le chemin complet (Domaine / Categorie / Element / ...)
   - Le nom en gras pour identification rapide

2. NAVIGATION VERS LES ELEMENTS LIES
   En cliquant sur un element/sous-element dans la liste des liaisons,
   on navigue directement vers son menu d'edition :
   - Changement automatique de domaine si necessaire
   - Selection de l'element ou sous-element
   - Ouverture de son panneau d'edition

3. DELIAISON INDIVIDUELLE
   Chaque element/sous-element dans la liste possede un bouton 
   de deliaison (icone LinkOff) visible au survol permettant de :
   - Couper la liaison avec cet element specifique seulement
   - Garder les autres liaisons intactes

4. BOUTON DELIAISON GLOBALE
   Le bouton "Delier cet element de tous" ou 
   "Delier ce sous-element de tous" reste disponible pour 
   couper toutes les liaisons en une seule action.

NOUVELLES FONCTIONS DANS LE STORE
---------------------------------
- getLinkedElements(elementId): Retourne tous les elements lies
- getLinkedSubElements(subElementId): Retourne tous les sous-elements lies
