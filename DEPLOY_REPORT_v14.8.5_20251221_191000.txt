CORRECTION APPORTEE
-------------------

DELIAISON DES SOUS-ELEMENTS LORS DU CHANGEMENT DE NOM DE L'ELEMENT

Scenario problematique :
1. Element "A" lie a un autre Element "A" dans un autre domaine
2. Duplication de l'un des "A" (les 3 sont lies)
3. Renommage de l'element duplique en "B" 
   -> L'element est delie (comportement attendu)
   -> MAIS les sous-elements gardaient leur ancienne liaison vers "A"
4. Liaison du nouvel element "B" a un autre element "B" existant
   -> Les sous-elements restaient lies a l'ancien groupe "A" au lieu de "B"

Solution :
Quand un element change de nom (ce qui le delie), tous ses sous-elements 
sont maintenant AUSSI delies automatiquement.

Ainsi, quand l'utilisateur lie ensuite le nouvel element a un autre groupe,
les sous-elements peuvent etre correctement lies au nouveau groupe 
(si l'option "Lier les sous-elements" est cochee).
