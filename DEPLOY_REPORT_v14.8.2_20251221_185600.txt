1. CRITICITE LA PLUS ELEVEE LORS DE LA LIAISON
   Quand on lie deux elements ou sous-elements :
   - Le statut le plus critique est applique aux deux
   - Ordre de criticite (du moins au plus critique) :
     ok < information < herite < deconnecte < mineur < critique < fatal

2. LIAISON AUTOMATIQUE DES SOUS-ELEMENTS DE MEME NOM
   Quand on lie deux elements avec l'option "Lier les sous-elements" :
   - Tous les sous-elements de meme nom deviennent lies
   - Ils prennent automatiquement le statut le plus critique

3. SYNCHRONISATION DES MODIFICATIONS
   Toutes les modifications sur un element/sous-element lie sont propagees :
   - Statut
   - Icones (icon, icon2, icon3)
   - Valeur et unite
   - Publiable (pour les elements)
   
   EXCEPTIONS (non propagees) :
   - Le nom (le changement de nom coupe la liaison)
   - Le domaine (chaque element reste dans son domaine)
   - La suppression (supprime uniquement l'element concerne, les autres restent lies)
