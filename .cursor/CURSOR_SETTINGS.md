# Configuration Cursor pour désactiver les confirmations automatiques

## Méthode 1 : Paramètres de l'Agent (Recommandé)

1. Ouvrez Cursor
2. Appuyez sur `Ctrl+,` (Windows) ou `Cmd+,` (Mac) pour ouvrir les paramètres
3. Recherchez "Agent" dans la barre de recherche
4. Activez l'option **"Application auto des modifications"** ou **"Auto-apply changes"**

Cela permettra à l'Agent d'appliquer automatiquement les modifications sans demander de confirmation.

## Méthode 2 : Utiliser le mode Plan

Au lieu d'utiliser le mode Agent, utilisez le mode **Plan** :
- Le mode Plan génère d'abord un plan complet de toutes les modifications
- Une fois le plan approuvé, toutes les modifications sont appliquées en une seule fois
- Aucune interruption supplémentaire

Pour activer le mode Plan :
1. Dans Cursor, sélectionnez le mode **Plan** au lieu du mode **Agent**
2. Laissez Cursor générer le plan complet
3. Approuvez le plan une seule fois
4. Toutes les modifications seront appliquées automatiquement

## Note

Ces paramètres sont des préférences de l'interface Cursor et ne peuvent pas être configurés via un fichier de configuration de projet. Vous devez les modifier dans les paramètres de Cursor directement.
