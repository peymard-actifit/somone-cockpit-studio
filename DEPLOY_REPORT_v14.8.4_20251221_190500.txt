PROPAGATION DE LA COULEUR VIA L'ELEMENT PARENT LIE

Probleme : Quand on modifiait la couleur (statut) d'un sous-element, 
les sous-elements lies dans d'autres elements ne se mettaient pas a jour.

Solution : La fonction updateSubElement() propage maintenant les modifications
via DEUX mecanismes :

1. LIAISON DIRECTE (linkedGroupId du sous-element)
   - Si le sous-element a un linkedGroupId, on propage aux sous-elements 
     qui partagent le meme linkedGroupId

2. LIAISON VIA L'ELEMENT PARENT
   - Si l'element parent est lie (a un linkedGroupId)
   - On trouve tous les elements lies au meme groupe
   - On cherche les sous-elements de MEME NOM dans ces elements
   - On propage les modifications a ces sous-elements

Cela garantit que meme si les sous-elements n'ont pas ete explicitement 
lies (par exemple, donnees anciennes ou liaison sans cocher "Lier les 
sous-elements"), les modifications sont propagees si leurs elements 
parents sont lies ET s'ils ont le meme nom.
